
## 安装

以学习语句为主，其他为辅；所以直接使用 `docker` 构建一个 `MySQL` 的容器进行学习；

```bash :no-line-numbers
docker run -d \
  --name mysql80 \
  -p 3306:3306 \
  -e MYSQL_ROOT_PASSWORD=123456 \
  -v mysql_data:/var/lib/mysql \
  mysql:8.0
```
随便找个客户端连接即可，或者使用命令行连接，但是要先进入容器；

## 统一字符编码

通过命令行进入后输入 `\s` 会显示字符编码；

```bash :no-line-numbers
mysql -u root -p

\s

# Server characterset:    utf8mb4
# Db     characterset:    utf8mb4
# Client characterset:    latin1
# Conn.  characterset:    latin1
```

通过 `SET NAMES utf8mb4;` 指令将字符集统一调整

## 库操作

```sql :no-line-numbers
-- 创建数据库名称为 db1 charset 指定字符编码
create database db1 charset=utf8;

-- 查所有数据库
show databases;
-- 查单个数据库，创建这个数据库时的 sql 语句
show create database db1;

-- 数据库只能修改字符编码
alter database db1 charset=gbk;

-- 删除数据库
drop database db1;
```

## 表操作

操作表之前要先知道表在哪个数据库中；进入到指定的数据库中进行操作。

```sql :no-line-numbers
select database(); -- 查看当前所在数据库
use db1; -- 切换到指定数据库
```

```sql :no-line-numbers
-- 创建表: 创建一个名为 movies 的表，里面包含两个字段
create table movies(id int, age char);

-- 查看所有表
show tables;
-- 查看表的创建语句
show create table movies;
-- 查看一张表的结构
describe movies; 
desc movies;

-- 修改表
-- 修改某个字段的字符长度
alter table movies modify age char(4);
-- 修改某个字段的名称
alter table movies change age Age char(5);

-- 删除表
drop table movies;
```

## 操作记录

```sql :no-line-numbers
-- 插入数据
insert into movies values(1, '流浪地球');
insert into movies values(1, '流浪地球'), (2, '三体'); -- 多条记录

-- 查询数据
select * from movies; -- * 标识所有
select id, name from movies; -- 查看指定字段

-- 修改数据
update movies set name='满江红' where id=1; -- 要有 where 增加限定条件不然就全改了

-- 删除数据
delete from movies where name='三体'; -- 要有 where 增加限定条件不然就清空了
```
