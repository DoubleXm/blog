## 用户管理

```sql
-- 查询用户: 用户数据在 mysql.user 表中
select * from mysql.user;

-- 创建用户
create user '用户名'@'主机名' identified by '密码';
-- 示例，如果不限制的登录的话主机名可以设置为 % 表示任一
create user 'xsq'@'localhost' identified by 'admin123';

-- 修改用户密码
alter user '用户名'@'主机名' identified with mysql_native_password by '新密码';
-- 修改主机名
update user set host='主机名' where user = '用户名';

-- 删除用户
drop user '用户名'@'主机名';
```

## 权限控制

默认情况下创建的用户只能看到一个系统库，没有其他任何权限（创建、删除、查询、修改）

通过 `select * from mysql.user;` 可以看到表中包含很多的权限，最常用的如下：

- `all、all privileges` 所有权限
- `insert` 插入数据权限
- `delete` 删除数据权限
- `update` 修改数据权限
- `select` 查询数据权限
- `create` 创建表权限
- `alter`  更新表权限
- `drop` 删除表权限

权限相关语法分类三类：查询、分配、删除

```sql
-- 查询
show grants for '用户名'@'主机名';
-- 示例 show grants for 'xsq'@'localhost'；

-- 分配
grant 权限列表 on 库名.表名 to '用户名'@'主机名';
-- 示例 grant all on db6.* to 'xsq'@'localhost';

-- 撤销
revoke 权限列表 on 库名.表名 from '用户名'@'主机名';
```

有时更改权限后如果没有效果可以使用 `flush provileges;` 刷新权限。
