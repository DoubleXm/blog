## 库的 CURD

```sql
show databases; -- 查看所有数据库
create database [库名]; -- 创建数据库
show create database [库名]; -- 查看数据库创建语句
alter database [库名] charset=utf8; -- 修改数据库编码
drop database [库名]; -- 删除数据库
use [库名]; -- 进入某个数据库
select database(); -- 查看当前所在数据库
```

## 查看表语法

```sql
show create table [表名]; -- 查看表的创建语法
-- 查看表结构
describe [表名];
desc [表名];
show tables; -- 查看所有表
```

## 创建表的语法

```sql
create table <表名> (
  <字段名1> <字段类型> [(宽度)] [约束条件1 约束条件2],
  <字段名2> <字段类型> [(宽度)] [约束条件]
);

-- 示例
create table movies (
  id int not null auto_increment, -- 自增且不为空
  name char(10) not null, -- 不能为空
  age int default 1, -- 默认值 1
  primary key (id) -- 将 id 设置为主键
);
```

## 常用的约束条件

其实 `ORM` 把这些事情都做了，不过还是熟悉一下常用的比较好；

通过 `desc` 查看表结构的时候会有一些字段会显示这些约束的条件；

```
+-------+---------------------------+------+-----+-----------+----------------+
| Field | Type                      | Null | Key | Default   | Extra          |
+-------+---------------------------+------+-----+-----------+----------------+
| id    | int                       | NO   | PRI | NULL      | auto_increment |
| port  | int(10) unsigned zerofill | YES  | MUL | NULL      |                |
| host  | varchar(15)               | NO   |     | 192.0.0.1 |                |
+-------+---------------------------+------+-----+-----------+----------------+
```

### type 类型约束

- `auto_increment` 自增
- `unsigned` 数字使用，标识使用无符号尽可能多约束条件的最前面
- `zerofill` 数字使用，用 0 填充；当给 `int` 类型设置长度时会有作用 `int(3)`，没啥用

```sql 
-- 自增字段应该是 primary key 或者 unique
create table t1 (id int primary key auto_increment, name char(10));

-- 不论数据是否被删除以及插入什么数据，自增数据都会以上一条数据为准开始自增
insert into t1(name) values ('tom'), ('jack');
```


### null

`not null` 不能为空

```sql
create table t1 (id int not null);
```

### key 约束

`primary key (fieldName)` 设置主键，约束特性：不为空且唯一 `unique + not null` 会被数据库识别为 `primary key`

```sql
create table t1 (id int primary key, name char(10));

-- 可以放在最后通过括号指定
create table t1 (id int, name char(10), primary key (id));

-- 也可以通过 primary key 指定多个字段，叫做复合主键
create table t1 (id int, host char(15), port int, primary key(host, port));
```

`unique` 全表数据唯一

```sql
create table t1 (id int unique, name char(10));

-- 可以放在最后通过括号指定
create table t9 (id int, name char(10), unique(id));

-- 联合唯一，多个字段不允许重现重复
-- host 可以重复、port 可以重复，但是不允许 host 和 port 同时重复
create table t10 (id int, host varchar(15), port int, unique(host, port));
```

### default 默认值

`default [value]` 不能为空

```sql
create table t1 (id int not null default 0);
```

## 修改表的语法

```sql 
-- 修改存储引擎
alter table [表名] engine=<存储引擎名字>;

-- 修改表名
alter table [表名] rename <新表名>;

-- 增加字段 first 第一个；after 某个字段后面
alter table [表名] 
  add <字段名><字段类型> [约束条件] [first | after<字段名>],
  add 字段2...;

-- 删除字段
alter table [表名] drop <字段名>;

-- 修改字段类型
alter table [表名] modify [字段名]<字段类型> [约束条件]

-- 修改字段名及类型
alter table [表名] change [旧字段名] [新字段名]<新字段类型> [约束条件]
```


## 删除和复制表语法

```sql
-- 删除表
drop table [表名]

-- 复制表
create table [表名] select * from <被复制表名> [条件];

-- 复制表结构
create table [表名] like <被复制表名>;
```
