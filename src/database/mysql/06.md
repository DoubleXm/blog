
## 字符串函数

```sql
select concat('1', '2', '3'); -- 拼接
select lower('HELLO'); -- 转小写
select upper('hello'); -- 转大写
select lpad('hello', 10, ',') -- 左填充 10 个逗号
select rpad('hello', 10, ',') -- 右填充 10 个逗号
select trim(' hello '); -- 去除两边空格
select substring('hello', 2, 3) -- 字符串切片
```

## 数值函数

```sql
select ceil(1.3); -- 向上去整
select floor(1.6); -- 向下去整
select mod(10, 2); -- 取模，10 % 2 余数
select rand(); -- 获取 0-1之间的小数
select round(3.1415926, 4); -- 四舍五入，保留四位小数
```

## 日期函数

```sql
select curdate(); -- 当前日期 2026-04-30
select curtime(); -- 当前时间 22:33:44
select now(); -- 当前日期时间 2025-04-30 22:33:44
select year('2025-04-30 22:33:44'); -- 获取 datetime 年份
select month('2025-04-30 22:33:44'); -- 获取 datetime 月份
select day('2025-04-30 22:33:44'); -- 获取 datetime 日期
select hour('2025-04-30 22:33:44'); -- 获取 datetime 小时
select minute('2025-04-30 22:33:44'); -- 获取 datetime 分钟
select second('2025-04-30 22:33:44'); -- 获取 datetime 秒
```

## 时间差计算
可以被 `date_add` 使用的的单位有 `year、month、day、hour、minute、second`

```sql
select date_add(now(), interval 1 year); -- 今年的当前时间加一年

select datediff('2026-05-30 21:51:00', now()); -- 获取时间差, 单位天
select timediff('2026-05-30 21:51:00', now()); -- 小时为单位
```

## 流程控制函数


```sql
-- gender 是 male 返回男，否则返回女， 类似三元表达式
select name, if (gender = 'male', '男', '女')from emp;

-- 如果 notes 不存在就返回 unknown
select name, ifnull(notes, 'unknown') from emp;
```

复杂的判断条件要使用 `case when ... then` 关键字

```sql
-- 查询所有员工
-- 大于 1w 显示核心员工
-- 大于 5k 普通员工
-- 大于 1k 新员工
select 
  name,
  case
    when salary > 1000 then '核心员工'
    when salary > 5000 then '普通员工'
    else '新员工'
  end as '员工分类'
from emp;

-- 也可以是
select 
  name,
  case dep
    when '技术部' then '高级技术顾问'
    when '人事部' then 'HR'
    else '销售部'
  end as 'dep_format'
from emp;
```
