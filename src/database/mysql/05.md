
## 增删改

```sql
-- 插入数据
insert into <表名> values (<值1>, <值2>, ...);
-- 插入指定数据
insert into <表名> (<字段名1>, <字段名2>, ...) values (<值1>, <值2>, ...);
-- 插入多行数据
insert into <表名> values 
  (<行1值1>, <行1值2>, ...),
  (<行1值1>, <行1值2>, ...);

-- 更新数据，没有 where 会统一修改表中的所有数据
update <表名> set <字段名1> = <新值1> [, <字段名2> = <新值2> ...] [where <条件>];

-- 删除数据，没有 where 表中数据会被清空
delete from <表名> [where <条件>];
```

## 单表查询

```sql
-- distinct 去重，去除的是查出来的数据重复
select distinct <字段...> from <库名>.<表名>
  where <条件>
  group by <分组条件>
  having <过滤条件>
  order by <排序字段> {ASC 升序 | DESC 降序} -- 默认升序
  limit n;
```

## 四则运算

查询出来的数据是支持数学运算的；比如查询所有员工的年薪；

```sql
select name, salary * 12 from emp;

-- 通过上面查询来的虚拟表，表头为 `name` `salary * 12` 这样显示很不直观；
-- 可以通过 as 给年薪设置一个合理的表头
select name, salary * 12 as year_salary from emp;
```

## 格式化显示 concat

```sql
-- 可以将数据查询出 姓名: xx年龄: xx | 薪资:xx 的这种结果，算是对结果的 format
select concat('姓名: ', name, '年龄: ', age), concat('薪资:', salary) from emp;
```

## where 过滤条件

`where` 后跟基本的运算表达式；

- `and` `or` 表示或和且的关系；
- `between` 表示某个数据的区间；`between 10 and 20`，这是闭合区间；前后包含。
- `in` 表示包含的关系；`age in (10, 20, 30)`
- `not` 表示否的关系；直接 `not in | not between | not like` 和运算表达式一起 `not (age > 10 as age < 20)`
- `is null | is not null` 对空的判断；
- `like` 模糊匹配 `_` 匹配一个字符 `%` 匹配多个字符

```sql
-- 查询大于 35 岁的 name 和 age
select name, age from emp where age > 35;

-- 多条件使用 and 进行拼接
select name, age from emp where dep = '技术部' and age > 35;

-- 也可以使用 between, 查询 age 20 到 40 之间的数据
select name, age from emp where age between 20 and 40; -- 闭区间：包含 20 和 40
-- 等价于
select name, age from emp where age >= 20 and age <= 40;

-- 此外逻辑关键字还有 not 和 or
select name, age from emp where age < 20 or age > 40;
-- 可以把 between x and y 当成一个固定公式，总是会一起出现
select name, age from emp where not age between 20 and 40;

-- 通过 in 判定符合一组条件的数据
select name, salary from emp where salary in (3000, 4000, 5000);

-- 通过 is null 或者 is not null 判断数据是否为 null
select * from emp where notes is null;
select * from emp where notes is not null;

-- like 模糊匹配
select name from emp where name like '荀_'; -- 所有荀开头的 2 个字符的 name 都会被查询
select name from emp where name like '丁_'; -- 所有丁开头的 name 都会被查询出来
```

## group by 分组条件

查询的数据只能是分组后的数据，主要的目的是为了对分组后的数据进行处理（聚合函数）。

聚合函数包含 `count | max | min | sum | avg`；

但是如果不使用 `group by` 默认就是一个分组，也可以直接使用这些聚合函数。

```sql
-- 根据 dep 进行分组后就只能查询 dep
select dep from emp group by dep;

-- 统计每个部分的员工数量
select dep, count(id) from emp group by dep;
-- 统计每个部分中年龄最大的员工
select dep, max(age) from emp group by dep;
-- 统计每个部分中薪资最低一个
select dep, min(salary) from emp group by dep;
-- 统计每个部分中的平均年龄
select dep, avg(age) from emp group by dep;
-- 统计每个部分的工资总和
select dep, sum(salary) from emp group by dep;
```
