## 存储过程

> [!INFO]
> 经过编译，并存储在数据库中的一段 SQL 集合；
>
> 就是 SQL 的封装和重用，调用存储过程可以简化业务端的工作，减少数据在客户端和服务端的传输从而提高效率。

```sql
-- 创建存储过程
create procedure 存储过程名称([参数...])
begin
    sql 语句
end;

-- 调用存储过程
call 存储过程名称()

-- 查看存储所有过程 infomation_schema.ROUTEINGS 查询可以带任何查询条件
select * from infomation_schema.ROUTEINES;

-- 查看存储过程
show create procedure 存储过程名称;

-- 删除存储过程
drop procedure if exists 存储过程名称;
```

## 命令行创建存储过程

由于存储过程中也包含一些 `sql` 语句导致直接出现多个分号会创建失败；

```sql
-- 使用 delimiter 修改结束符
delimiter $$
create procedure p1()
begin
    select * from emp;
end;
delimiter $$
```

## 变量

- 系统变量 `global | session`

```sql
-- 查看系统变量
select @@[global | session] 系统变量名;

show [global | session] variables;
show [global | session] variables like '';

-- 设置系统变量
set @@[global. | session.]系统变量名 = 值;
set [global. | session.]系统变量名 = 值;
```

- 用户自定义变量

```sql
-- 定义变量
set @变量名 = 值[,...];
set @变量名 := 值[,...];
-- select 只能使用 :=
select @变量 := 值[,...];
-- info 则是将查询的数据赋值给这个变量
select 字段名 into @变量名 from 表名 where 条件...;

-- 查询变量
select @变量名;
```

- 局部变量

在存储过程中定义的变量，必须在 `begin` 后面 `sql` 语句前面。存储过程中的变量不需要增加 `@`

```sql
delcare 变量名 变量类型 [default 默认值];

create procedure p1()
begin
  declare num int default 0;
  select count(*) into num from emp;
  select num;
end;

call p1();
```

## 条件判断

单分支 `if` 判断

```sql
create procedure p1()
begin
  declare gender varchar(16) default 'male';
  set gender = if (gender = 'male', '男', '女');
  select gender;
end;
call p1();
```

多分支 `if` 判断

```sql
delimiter $$
create procedure p1()
begin 
  declare score int default 80;
  declare notice varchar(10) default '';
  if (score = 100) then
    set notice = '优秀';
  elseif (score = 80) then
    set notice = '不错';
  else
    set notice = '垃圾';
  end if;
  select notice;
end $$
delimiter;
call p1();
```

多分支 `case` 表达式

```sql
delimiter $$
create procedure p1()
begin
  declare score int default 80;
  declare notice varchar(10) default '';
  set notice = case score
    when 100 then '优秀'
    when 80 then '不错'
    else '垃圾'
  end;
  select notice;
end $$
delimiter;
call p1();
```

多分支 `case` 语句

```sql
delimiter $$
create procedure p1()
begin
  declare score int default 80;
  declare notice varchar(10) default '';
  set notice = case
    when score >= 90 then '牛逼'
    when score >= 80 then '还行'
    else '垃圾'
  end;
  select notice;
end $$
delimiter;
call p1();
```
